Sobre mim

Vinicius Naziozeno Santoro do Rio

Olá, sou o Vinicius, tenho 21 e estou cursando o terceiro semestre de ciência de dados na Faculdade de Tecnologia de São Paulo (FATEC).

Trabalho no Itaú Unibanco, desenvolvendo modelagem de dados através do Alteryx e Python e construindo indicadores via Tableau/Power BI. Com conhecimento adquirido na formação de engenharia de software pela 42 São Paulo, possuo experiência com versionamento de código com Git e GitHub.

Sabendo das tendências para a cloud, me certifiquei pela AWS (CLF-C01).

Estudo diariamente para ser capaz de resolver problemas complexos de forma autônoma, criativa e colaborativa. Além disso, tenho total comprometimento com autoconhecimento e respeito nos relacionamentos interpessoais.
